# movie-app-ios

An app for discovering new movie premieres and adding notifications for them.

## Required libraries

- Cocoapods
- SwiftGen

## Installation

Install movie-app-ios with **pod**

```bash
  pod install
```

Update assets with **swiftgen**

```bash
  swiftgen
```

Then open it with file **movie-app-ios.xcworkspace**
